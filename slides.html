prezto

A simple slideshow system with HTML+CSS+JS

Press space or →

You can go back with ←

home jumps to the first slide

Goals

  • dead simple
  • no bloat
  • use web standards
  • near-zero requirements

Requirements

  • a text editor
  • a web server (optional)
  • Zepto.js

Features

  • split your HTML document in slides
  • navigate with your keyboard
  • extensible through JS
